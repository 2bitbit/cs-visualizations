PC相对寻址计算 (R_X86_64_PC32):
CSAPP公式: *refptr = ADDR(r.symbol) + r.addend - refaddr
1. 目标符号 `swap` 运行时地址 ADDR(r.symbol) = 0x400010
2. 占位符运行时地址 refaddr = 0x400005
3. 重定位条目的加数 r.addend = -4
4. 计算: 0x400010 + (-4) - 0x400005
   = 0x40000C - 0x400005 = 0x7
注：由于 x86 架构特性，这等效于直观的 "目标地址(0x400010) - 下一条指令PC(0x400009) = 0x7"
5. 填入 0x400005 处: 07 00 00 00 (小端序)
绝对寻址重定位 (R_X86_64_64):
CSAPP公式: *refptr = ADDR(r.symbol) + r.addend
1. 引用的符号 `buf` 地址 ADDR(r.symbol) = 0x600000
2. 重定位加数 r.addend = 0
3. 计算: 0x600000 + 0 = 0x600000
4. 填入 0x600008 的位置: 00 00 60 00 00 00 00 00 (小端序)
