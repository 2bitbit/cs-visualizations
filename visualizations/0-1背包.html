算法经典：0-1 背包问题

问题描述

有一个容量为 W = {{ backpackCapacity }} 的背包。 现有 N = {{ items.length }} 个物品，每个物品有一个重量和一个价值。

"0-1" 的含义：每个物品只有两种选择：放入背包(1) 或 不放入背包(0)。不能只放物品的一部分。

目标：在不超过背包容量的前提下，装入的物品总价值最大。

核心：状态转移方程

设 dp[i][j] 表示：前 i 个物品，放入容量为 j 的背包中，所能获得的最大价值。

初始状态：等待开始...
对于当前物品 i={{currentI}}, 重量 w={{items[currentI-1].weight}}, 价值 v={{items[currentI-1].value}}, 当前背包容量 j={{currentJ}} :
1. 若 j < w (当前容量 {{currentJ}} < {{items[currentI-1].weight}}，装不下):
   dp[i][j] = dp[i-1][j]
2. 若 j ≥ w (当前容量 {{currentJ}} ≥ {{items[currentI-1].weight}}，可以装):
   dp[i][j] = max(
     dp[i-1][j], // 不装该物品
     dp[i-1][j-w] + v // 装入该物品
   )
